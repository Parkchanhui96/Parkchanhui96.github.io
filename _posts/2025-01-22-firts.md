# 완전정복 9가지만 알면면 충분하더라.

***


## 1. 헤더(Header)
   
 : 제목 만들 때 쓰면 좋다.

### Syntax (입력규칙)

<pre>
#  H1 헤더 (최상위 제목)
##  H2 헤더 : (부제목)
###  H3 헤더 : (소제목)
</pre>

### Example (실행예시)

#  H1 헤더 (최상위 제목)
##  H2 헤더 : (부제목)
###  H3 헤더 : (소제목)




## 2. 굵은 글씨(BOLD):

: 텍스트를 강조하고 싶을 때 쓴다

  - 두 개의 별표(**) OR 밑줄(__)로 감싼다.

### Syntax (입력규칙)

<pre>
**굵은 글씨**
__굵은 글씨__
</pre>

### Example (실행예시)


**굵은 글씨**
__굵은 글씨__



## 3. 기울임 글씨 (ltalic)

: 인용문의 강조나 강조의 변주를 주고 싶을 때 쓰자.

  - 하나의 별표(*) OR 밑줄(_)로 감싼다.

### Syntax (입력규칙)

<pre>
*기울임 글씨*
_기울임 글씨_
</pre>
  
### Example (실행예시)  
  
*기울임 글씨*  
_기울임 글씨_  
  
  
  
  
## 4. 리스트(List)   
  
: 순서 없는 글의 가독성을 높이기 위해 분류할 떄 쓰자  
  
  - 순서 없는 리스트(Unordered List) 항목 앞에 (-), (+), (*) 를 사용한다  
  
### Syntax (입력규칙)  
  
<pre>  
- 항목 1  
- 항목 2  
</pre>  
  
### Example (실행예시)  
  
- 항목 1  
- 항목 2  
    
  - 순서 있는 리스트 (Ordered List): 숫자와 점(.)을 사용하여 항목을 나열한다
  
<pre>
1. 첫 번째 항목
2. 두 번째 항목
</pre>
  
### Example (실행예시)
  
1. 첫 번째 항목  
2. 두 번째 항목  
  
  
  
## 5. 링크(Links) 
  
  - <pre>링크는 [링크 텍스트](URL) 형식으로 작성합니다.</pre>
  
### Syntax (입력규칙)
  
<pre>
[Google](https://www.google.com)
</pre>
  
### Example (실행예시)
  
[Google](https://www.google.com)
  
  
  
## 6. 이미지(Images):
  
  - 링크와 비슷하지만 앞에 !을 추가하는 게 다르다
  
### Syntax (입력규칙)
  
<pre>
![이미지 설명](https://www.example.com/image.jpg)
</pre>
  
### Example (실행예시)
  
![이미지 설명](https://www.example.com/image.jpg)
  
  
  
## 7. 인용문(Blockquote)
  
  - 인용문은 > 기호로 작성합니다.
  
  
### Syntax (입력규칙)
  
<pre>
> 이것은 인용문입니다.  
> 이것은 인용문입니다.  
> 이것은 인용문입니다.  
> 이것은 인용문입니다.  
</pre>
  
### Example (실행예시)

> 이것은 인용문입니다.  
> 이것은 인용문입니다.  
  > 이것은 인용문입니다.  
  > 이것은 인용문입니다.  
  
  
  
## 8. 취소선(Strikethrough)

  : 잘못된 말이나, 글을 수정해야 할 때 쓰자

        - 양 끝단에 (~~)를 붙여 표시한다.

### Syntax (입력규칙)

<pre>
~~이것은 취소선 입니다.~~
</pre>

### Example (실행예시)

~~이것은 취소선 입니다.~~





## 9. 수평선(Horizontal Rule)

  : 글을 분류해 가독성을 높일 때 쓰자.
  
        - 수평선은 ---, ***, 또는 ___으로 생성할 수 있습니다.

### Syntax (입력규칙)

<pre>
***
___
</pre>

### Example (실행예시)

---
***
___




## 10. 코드(Code):


    - <pre>텍스트 내에서 코드나 명령어를 표시하려면 텍스트를 세 개의 백틱(```)으로 감쌉니다</pre>
    

### Syntax (입력규칙)

<pre>
```
java
// 코드가 들어갈 공간입니다.
```
</pre>

### Example (실행예시)

```
 java
// 코드가 들어갈 공간입니다.
```

    -  텍스트의 공백, 줄 바꿈, 탭 등을 그대로 유지하여 원본 그대로의 서식을 보여줍니다.

### Syntax (입력규칙)


<pre><pre>
텍스트의
  줄
바꿈을
      해보고 싶을 때.
</pre></pre>


    - pre는 서식이 있는 텍스트를 표시하는 태그입니다. <pre> 태그로 감싸져 있는 영역이 서식이 유지된 텍스트를 표시하고, <code> 태그는 코드 조각을 구분합니다

### Example (실행예시)

<pre>
텍스트의
  줄
바꿈을
      해보고 싶을 때.
</pre>
